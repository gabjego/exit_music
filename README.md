Exit Music - Premium EPK

Bienvenido al repositorio del Electronic Press Kit (EPK) digital de Exit Music, el show tributo a Radiohead basado en Guadalajara, Jalisco.

🎸 Sobre el Proyecto

Este EPK es un sitio web estático interactivo diseñado para proyectar profesionalismo y "caos artístico" a promotores, gerentes de hoteles de lujo y organizadores de festivales.

Su diseño está fuertemente inspirado en la estética técnica y fragmentada de la era OK Computer y Kid A (con referencias a errores de sistema, códigos de barras y líricas ocultas), manteniendo una usabilidad impecable para visualizar el rider técnico y el contacto.

🛠️ Cómo personalizar el código

El archivo principal es index.html. No requieres conocimientos avanzados para editarlo:

Reemplazar Fotos: Busca las etiquetas <img> en el código. Actualmente usan imágenes de prueba (images.unsplash.com...). Solo tienes que subir tus propias fotos a este repositorio y cambiar la dirección por el nombre de tu foto (ej. foto-banda.jpg).

Actualizar Historial:
Si tocan en un lugar nuevo, busca la sección id="venues" y cambia el nombre de uno de los lugares anteriores por el nuevo recinto.

Contacto:
Los números de WhatsApp y enlaces a Facebook están en la sección id="booking".
